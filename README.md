## 🚀 About Project

**UCAN Disable Thermal XPro** adalah modul Android untuk menonaktifkan sistem **thermal throttling** bawaan perangkat.

Dengan menonaktifkan thermal limit, perangkat dapat mempertahankan performa **CPU / GPU lebih stabil saat gaming atau heavy task**.

Project ini merupakan bagian dari **UCAN Project**.

## ⚡ Features

✔ Universal Android thermal disabler
✔ Stop multiple thermal services
✔ Disable kernel thermal limits
✔ Improve gaming performance
✔ Reduce CPU throttling
✔ Compatible with Magisk & KernelSU
✔ Auto device information during install

---

## 📱 Device Information (Installer)

Saat install modul akan menampilkan informasi:

* Device Name
* Android Version
* Kernel Version
* ROM Info
* Brand
* Model
* Board
* System Date

---

## 📦 Installation

1. Download **latest release**
2. Open **Magisk App / KernelSU**
3. Go to **Modules**
4. Install module ZIP
5. Reboot device

---

## ⚠️ Warning

Disabling thermal protection may cause:

* Device temperature increase
* Higher battery usage
* Potential hardware stress

Use at **your own risk**.
